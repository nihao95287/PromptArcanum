# 🚀 PromptArcanum (AI 提示词管理工作台)

## 📖 项目简介
PromptArcanum 是一个轻量级的 AI 提示词（Prompt）管理工具。本项目基于面向对象思想开发，旨在帮助用户在终端控制台中高效地记录、管理和回溯与大语言模型交互时使用的各种提示词及高级配置参数。

## ✨ 核心功能
* **📝 创作新提示词**：支持结构化录入核心提示词内容以及独立的附加参数（如 `--v 5 --ar 16:9`）。
* **🕰️ 历史记录回溯**：自动记录每次生成的提示词及时间戳，提供清晰的控制台列表展示，方便复用和二次优化。

## 🛠️ 技术栈
* **开发语言**: Java
* **开发环境**: IntelliJ IDEA
